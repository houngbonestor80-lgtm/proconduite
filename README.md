# ProConduite

Site vitrine pour une auto-école, développé avec **Laravel 9** et **Tailwind CSS**.

## Pages

- **Accueil** — présentation de l'auto-école
- **Services** — formations proposées (code, conduite, permis accéléré, etc.)
- **Tarifs** — grille tarifaire des formules
- **À propos** — présentation de l'équipe et de l'établissement
- **Contact** — formulaire de contact

## Stack technique

- [Laravel 9](https://laravel.com) (PHP 8)
- [Tailwind CSS](https://tailwindcss.com)
- Vite

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
npm run build
php artisan serve
```
